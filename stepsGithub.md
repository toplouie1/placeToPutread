## steps to push code to Github

- Create Folder
`mkdir nameOfFolder`
- Navigat into newly creater folder 
`cd namOfFolder`
- Create Local Repository 
`git init `
- Create code
`touch readme.md`

- SAVE CODE 

- Track/Stage/Watch
```
git add .
git add -A
git add readme.md

```
- commit out changes to our local git 
```
git commit -m "First commit"

```
